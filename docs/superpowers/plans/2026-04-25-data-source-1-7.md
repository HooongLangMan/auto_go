> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a stable 1.7 data-source layer that enriches drafts with standings/rank context and recent form without blocking the existing content pipeline.

**Architecture:** Keep the current crawler/router/content flow, but insert a structured enrichment layer that prefers API-Football and football-data, then falls back to OpenFootball and ClubElo without confusing Elo rank with league standings. Persist the richer structured fields in `MatchInfo` and build lightweight summaries for content generation.

**Tech Stack:** Python 3.13, Pydantic, SQLAlchemy, Typer, pytest, httpx
